Historia dos computadores

(Voltar)

O primeiro computador foi criado entre 1943 e 1946 e se chamava Eric Eletronic Numerical Integrator and Computer - ENIAC. Para o seu desenvolvimento, foi investido cerca de 6 milhões de dólares e seu funcionamento dependia de uma série de fatores, como 200 mil watts de energia. Um fator de grande importância é a memória, algo que não tinha na época do ENIAC. Cerca de 10 anos depois, foi lançado o RAMAX 305, uma máquina de armazenamento em disco feito pela IBM. No começo, só empresas e certas instituições faziam uso de um computador e, com o passar do tempo, essa tecnologia seria acessada por todos.

O que são clientes

(Voltar)

O cliente de TI é o dono da demanda , ou seja , ele demanda algum tipo de serviço para o servidor. Normalmente essa solicitação é feita através de uma rede.

O que são servers

(Voltar)

Um servidor é um recurso dentro de um sistema computacional, capaz de processar aplicações, armazenar dados e prestar serviços, enquanto o cliente é um computador que solicita esses serviços e recursos.

Como é a estrutura básica de um html

(Voltar)

O documento HTML sempre inicia com o que chamamos de estrutura básica. Esta estrutura é quase que imutável. Essa estrutura contém o doctype, que é o que diz para o navegador que é uma página html. Também temos a tag body, que é onde fica o conteúdo em sim da página e a tag head(cabeçalho) que é onde ficam algumas instruções que são processadas antes da abertura da página.

Entendendo a diferença entre a listas ordenadas e não ordenadas

(Voltar)

Lista é um importante recurso de HTML, pois permite criarmos tópicos de textos para uma melhor exemplificação de um determinado assunto. São recursos extremamente usados, inclusive quando nem imaginamos que ele está sendo usado, como no caso de menus. Hoje, boa parte dos menus em HTML é feito com listas. Listas não ordenadas são as listas não enumeradas. São usadas para listar itens sem necessidade de estar em sequência. Já as listas ordenadas são enumeradas e geralmente possuem necessidade de estar em sequência.
